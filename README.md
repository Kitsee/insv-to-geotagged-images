# insv-to-geotagged-images
Simple script to generate geotagged images from a Insta 360 .insv file

## Description

## Getting Started

required installed
* ffmpeg
* exiftool

```
convert.py <path> <yaw>
```

## Upload
use the following command to upload
```
panoramax_cli upload --api-url https://www.panoramax.wales/ --duplicate-distance 4 --no-split <Files>
```
